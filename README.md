# 👻 casper
A lightweight, fast, and friendly message broker written in pure Go.

## Design Goals

- ⚙️ Zero configuration
- 📦 No external dependencies
- 📉 Low resource usage
- 👷 Low maintenance
- 🚀 Designed for both low-latency and high-throughput use cases
- 🖥 Optimised for small clusters
- 💕 Intuitive semantics
